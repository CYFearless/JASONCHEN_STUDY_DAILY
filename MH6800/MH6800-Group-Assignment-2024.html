Q1: Find Estimators σ~2 and σ^2 for the Variance σ2

i) Method of Moments:

  1. First Moment (Mean): We know that the mean of the distribution is μ0.

  2. Second Moment: The second moment of the population distribution is E[X2]=σ2+μ02.

  3. Sample Second Moment: The sample second moment is given by 1n∑i=1nXi2.

    To derive the estimator σ~2, equate the population second moment to the sample second moment: 1n∑i=1nXi2=σ2+μ02 Solving for σ2, we get: σ~2=1n∑i=1nXi2−μ02

ii) Maximum Likelihood Estimation (MLE):

  1. Likelihood Function: Given the sample X1,X2,…,Xn from a normal distribution N(μ0,σ2), the likelihood function is: L(σ2)=∏i=1n12πσ2exp⁡(−(Xi−μ0)22σ2)

  2. Log-Likelihood Function: Taking the natural logarithm, we obtain the log-likelihood function: log⁡L(σ2)=−n2log⁡(2πσ2)−12σ2∑i=1n(Xi−μ0)2

  3. Derivative and Maximization: Taking the derivative with respect to σ2 and setting it to zero: ∂log⁡L(σ2)∂σ2=−n2σ2+12σ4∑i=1n(Xi−μ0)2=0; Solving for σ2, we get: σ^2=1n∑i=1n(Xi−μ0)2

Q2: Determine if Estimators σ~2 and σ^2 are Unbiased

An estimator is unbiased if its expected value equals the parameter it estimates like E[σ~]=σ.

i) Unbiasedness of σ~2:

E[σ~2]=E[1n∑i=1nXi2−μ02]=1n∑i=1nE[Xi2]−μ02=1n∑i=1n(σ2+μ02)−μ02=σ2

Thus, σ~2​ is unbiased.

Q5: Construct a 95% Confidence Interval for σ2

We use the chi-square distribution for this purpose.

The pivotal quantity is: (n−1)σ^2σ2∼χn−12; then the 95% confidence interval for σ2 is: ((n−1)σ^2χ0.975,n−12,(n−1)σ^2χ0.025,n−12)​

Numerical Question

a) Simulate a Random Sample

We need to simulate a random sample of size n=100 from the N(μ0,σ2) distribution. We'll assume values for μ0 and σ2 to proceed. For this example, let's take μ0=0 and σ2=4, which corresponds to a standard deviation σ=2.

Python Code:

​x
import numpy as np


# Set the parameters
mu_0 = 0
sigma = 2
n = 100


# Generate the random sample
sample = np.random.normal(loc=mu_0, scale=sigma, size=n)

b) Compute Three Different Confidence Intervals

Using the sample data generated, we compute three different confidence intervals based on the pivots discussed earlier.

  1. Pivot 1: (n−1)σ^2σ2∼χn−12 The confidence interval is CI1=((n−1)σ^2χ0.975,n−12,(n−1)σ^2χ0.025,n−12)

  2. Pivot 2: σ~2σ2∼χn2 The confidence interval is: CI2=(σ~2χ0.975,n2,σ~2χ0.025,n2)

  3. Pivot 3: n(X―−μ0)2σ2∼χ12 The confidence interval is: CI3=(n(X―−μ0)2χ0.975,12,n(X―−μ0)2χ0.025,12)

Python Code:

xxxxxxxxxx
import scipy.stats as stats


# Calculate sample mean and variance
sample_mean = np.mean(sample)
sample_variance = np.var(sample, ddof=0)  # MLE variance


# CI 1: Pivot 1
ci1_lower = (n-1) * sample_variance / stats.chi2.ppf(0.975, df=n-1)
ci1_upper = (n-1) * sample_variance / stats.chi2.ppf(0.025, df=n-1)


# CI 2: Pivot 2
ci2_lower = sample_variance / stats.chi2.ppf(0.975, df=n)
ci2_upper = sample_variance / stats.chi2.ppf(0.025, df=n)


# CI 3: Pivot 3
ci3_lower = n * (sample_mean - mu_0)**2 / stats.chi2.ppf(0.975, df=1)
ci3_upper = n * (sample_mean - mu_0)**2 / stats.chi2.ppf(0.025, df=1)


# Print the results
print(f"CI 1: ({ci1_lower:.4f}, {ci1_upper:.4f})")
print(f"CI 2: ({ci2_lower:.4f}, {ci2_upper:.4f})")
print(f"CI 3: ({ci3_lower:.4f}, {ci3_upper:.4f})")

  • CI 3: While the interval might cover the true variance due to its massive length, its lack of precision makes it practically useless.

From this analysis, CI 1 (Pivot 1) is the best choice among the three.
